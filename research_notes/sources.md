# RAG 技术资料来源

本教程按截至 2026-05 能查到的公开论文、项目和安全资料整理。下面列出主线来源，正文会引用对应技术点。

## 基础与综述

- Lewis et al., 2020, Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks: https://arxiv.org/abs/2005.11401
- Gao et al., 2023, Retrieval-Augmented Generation for Large Language Models: A Survey: https://arxiv.org/abs/2312.10997
- A Survey on Knowledge-Oriented Retrieval-Augmented Generation, 2025: https://arxiv.org/abs/2503.10677
- Retrieval-Augmented Generation: A Comprehensive Survey of Architectures, Enhancements, and Robustness Frontiers, 2025: https://arxiv.org/abs/2506.00054

## 检索、改写与排序

- BEIR, 2021, heterogeneous zero-shot IR benchmark: https://arxiv.org/abs/2104.08663
- HyDE, 2022, Precise Zero-Shot Dense Retrieval without Relevance Labels: https://arxiv.org/abs/2212.10496
- Query2doc, 2023, Query Expansion with Large Language Models: https://arxiv.org/abs/2303.07678
- Step-back Prompting, 2023: https://arxiv.org/abs/2310.06117
- Query Expansion by Prompting Large Language Models, 2023: https://arxiv.org/abs/2305.03653
- Lost in the Middle, 2023, long-context position sensitivity: https://arxiv.org/abs/2307.03172
- Pairwise Ranking Prompting, 2023: https://arxiv.org/abs/2306.17563

## 高阶 RAG

- Self-RAG, 2023, learning to retrieve, generate, and critique: https://openreview.net/forum?id=jbNjgmE0OP
- Corrective RAG, 2024: https://arxiv.org/abs/2401.15884
- RAPTOR, 2024, Recursive Abstractive Processing for Tree-Organized Retrieval: https://arxiv.org/abs/2401.18059
- Microsoft GraphRAG project, 2024-2025: https://www.microsoft.com/en-us/research/project/graphrag/
- LightRAG, 2024, graph-structured lightweight RAG: https://arxiv.org/abs/2410.05779

## 评估与安全

- ARES, 2023, automated RAG evaluation: https://arxiv.org/abs/2311.09476
- RAGBench, 2024, explainable benchmark for RAG systems: https://arxiv.org/abs/2407.11005
- PoisonedRAG, 2024, knowledge corruption attacks: https://arxiv.org/abs/2402.07867
- BadRAG, 2024, vulnerabilities in retrieval augmented generation: https://arxiv.org/abs/2406.00083
- OWASP Top 10 for LLM Applications 2025: https://owasp.org/www-project-top-10-for-large-language-model-applications
- SD-RAG, 2026, prompt-injection-resilient selective disclosure: https://arxiv.org/abs/2601.11199
